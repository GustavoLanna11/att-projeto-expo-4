# att-projeto-expo-4

Projeto Expo que exibe uma lista de produtos utilizando a API Fake Store API.

## Funcionalidades

- FlatList com produtos da API https://fakestoreapi.com/products
- Cards de produtos com:
  - Imagem do produto
  - Título (máximo 2 linhas)
  - Preço
  - Descrição (máximo 2 linhas)
  - Categoria
  - Avaliação (nota e quantidade de avaliações)

## Como executar

1. Instale as dependências:
```bash
npm install
```

2. Inicie o projeto:
```bash
npm start
```

3. Escaneie o QR code com o app Expo Go ou pressione:
   - `a` para Android
   - `i` para iOS
   - `w` para Web
